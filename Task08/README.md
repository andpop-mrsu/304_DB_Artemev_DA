# Лабораторная работа 8. CRUD-приложение для работы с БД

Веб-приложение для ввода, сохранения в базе данных и вывода информации о составе групп и сданных экзаменах.

## Структура проекта

```
Task08/
├── public/          # Корень сайта
│   ├── index.php   # Главная страница со списком студентов
│   ├── student_*.php  # Файлы для работы со студентами
│   └── exam_*.php     # Файлы для работы с экзаменами
├── data/           # База данных
│   └── database.db # SQLite база данных (создается автоматически)
├── config/         # Конфигурация
│   └── database.php # Подключение к БД
└── php/            # PHP интерпретатор (для запуска на другом компьютере)
```

## Установка и запуск

### 1. Инициализация базы данных

Перед первым запуском необходимо инициализировать базу данных:

**Windows:**
```bash
cd Task08
init_database.bat
```

**Или вручную:**
```bash
cd Task08
php data/init_db.php
```

Скрипт создаст базу данных `data/database.db` из файла `db_init.sql`.

### 2. Запуск веб-сервера

Находясь в каталоге `Task08`, запустите встроенный PHP сервер:

```bash
php -S localhost:3000 -t public
```

Или используя PHP из папки `php`:

```bash
php\php.exe -S localhost:3000 -t public
```

### 3. Открытие приложения

Откройте в браузере:
- http://localhost:3000/index.php
- или просто http://localhost:3000/

## Функционал

### Работа со студентами

- **Главная страница**: Список всех студентов с сортировкой по номеру группы и фамилии
- **Фильтрация**: По номеру группы через выпадающий список
- **Добавление**: Форма для ввода нового студента
- **Редактирование**: Форма для изменения данных студента
- **Удаление**: Подтверждение и удаление студента со всеми связанными записями

### Работа с экзаменами

- **Список экзаменов**: Для каждого студента отображаются все экзамены в хронологическом порядке
- **Добавление**: Форма для ввода нового экзамена с выбором дисциплины на основе направления студента
- **Редактирование**: Форма для изменения данных экзамена
- **Удаление**: Подтверждение и удаление записи об экзамене
- **Ввод задним числом**: Возможность ввода результатов экзаменов за любой учебный год, в котором учился студент

## Технические детали

- **База данных**: SQLite
- **Подключение**: PDO с подготовленными запросами
- **Синтаксис PHP**: Альтернативный синтаксис (без echo), режим шаблонизатора
- **Веб-сервер**: Встроенный PHP сервер на порту 3000

## Требования

- PHP 8.0 или выше
- Расширение PDO SQLite

## Примечания

- Папка `php` содержит PHP интерпретатор для запуска на другом компьютере без установленного PHP
- База данных создается автоматически при первом запуске скрипта инициализации
- Все данные хранятся в файле `data/database.db`
